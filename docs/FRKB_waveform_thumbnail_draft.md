# 波形缩略图草案（需求版）

## 目标
- 新增一个波形样式：RGB
- 仅用于播放器内全曲波形展示
- 视觉效果严格复刻 Mixxx 的 RGB 波形
- 不做持久化

## 范围
- 仅播放器页面
- 不影响现有 SoundCloud / Fine 样式
- 不修改曲库数据结构与数据库

## 样式需求
- 样式名称：RGB
- 参考标准：Mixxx 的 RGB 波形表现为唯一标准
- 颜色、亮度、对比度、透明度等视觉表现需与 Mixxx 2.5.4 一比一一致
- 支持半波形 / 全波形显示模式
- 进度高亮、光标与交互保持现有体验一致

## 生成策略
- 波形数据由 Rust 模块生成
- 渲染层仅负责绘制

## 性能与体验
- 生成过程不应造成明显卡顿
- 频繁切歌时保持流畅

## 配置与交互
- 设置中可选择 RGB 样式
- 接入点为“波形样式”下拉框
- 切换样式应即时生效

## 验收标准
- 同一音频输入下，RGB 样式与 Mixxx RGB 波形视觉严格一致
- 播放器内正常显示、进度跟随正确
- 无持久化文件或数据库写入

## 验收基准
- 以 Mixxx 的输出为唯一真值
- Mixxx 版本：2.5.4（默认皮肤与配色）
- 对照样本在实现阶段指定（1-3 个固定音频）
