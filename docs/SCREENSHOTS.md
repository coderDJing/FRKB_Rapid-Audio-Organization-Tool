## 截图要求

### 文件格式
- **格式**: WebP（推荐）或 PNG
- **命名规则**:
  - 夜间模式：`功能名.webp` 或 `功能名_cn.webp`
  - 日间模式：`功能名_light.webp` 或 `功能名_cn_light.webp`
- **存放路径**: `docs/public/assets/`

### 尺寸要求
- **宽高比例**: 16:10（推荐）
- **建议范围**：主界面截图宽度 1000-1600px
- **一致性**：中文/英文、日间/夜间保持相同尺寸与视角

### 拍摄要求
1. 夜间模式使用暗色主题
2. 日间模式使用亮色主题
3. 确保界面清晰、无个人信息
4. 窗口大小适中，背景干净
